- External Files:

Scripts can also be placed in external files. It's more practical to use external scripts when some code is used in many different web pages. Javascript files have the file extension ' .js '. To use an external script, put the name of the script file in the src (source) attribute of the script tag. The .js file can't contain the script tag, just write javascript normally as if you were already in the <script> brackets </script> .

Example file name : <script src="myjs.js"> </script>

By clicking the button below, a javascript alert window will open. The javascript is in an external file.

Scope:

Similar to most languages:

- Local variables can only be used where declared.

- Global variables can be used inside functions, scripts, and the total JS environment.

- Assigning a value to variable that hasn't been declared, makes it Global.

- Don't create Global variables, unless you intend to.

- In HTML, the global scope is the window object, all global variables belong to the window class.

- Lifetime of variables: Deleted when function is completed, window is closed, but remains available to new pages loaded into the same window

JSON - JavaScript Object Notation :

Format for storing and transporting data, often used when data is sent from a server to a web page.

Lightweight data interchange format, independent, and easy to understand.

  • Data (Name and a value) : "name":"value" (Require double quotes "")
  • Objects : {"name":"value", "name":"value"}
  • Arrays : [ { "name":"value", "age":"value"}, { "name":"value", "age":"value"} ]
